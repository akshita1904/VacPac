Types of Vaccines




  • Living weakened organisms
  • Living organisms
  • Killed germs
  • Toxoids





Living Weakened Organisms: This set of vaccine is made weakening the microorganisms that cause the disease. Eg-Vaccine against tubercolosis





Living Organisms: This set of vaccines is made up of live and similar organisms. Eg-Vacccine against Smallpox





Killed germs: This set of vaccine is made using the dead organisms causing the disease. Eg-Vaccine against Polio





Toxoids: This set of vaccines is made from the toxins released from the organisms. Eg-Vaccine against Tetanus.
